# Bakery-Management
My first Python project (2023) - A command-line bakery management system with admin and customer interfaces.

## Features

### **Admin Access** (Password: `ready_go`)
-  Add new items to menu
-  Remove items from menu
-  Update product prices
-  View complete menu
-  View sales statistics with graphs
